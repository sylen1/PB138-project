Spojazdnenie backend-u

Pre backend som sa rozhodol použit framework Spring Boot, kvôli jednoduchému použitiu a takmer žiadnej konfigurácie potrebnej k spusteniu aplikácie. Pre spojazdnenie bolo potrebné iba vytvorit projekt na start.spring.io, vybrať potrebné moduly (v tomto prípade Web a Thymeleaf) a importovať do vývojového prostredia. Thymeleaf je nástroj pomocou ktorého je možné dynamicky generovať HTML pridaním premenných z Java kódu (alternatíva ku JSP).

Web Design

Pre krajší vzhľad bol použitý CSS framework Bootstrap 4 a ikonky použté z FontAwesome. Pri návrhu stránok som sa snažil vytvoriť moderný, minimalistický vzhľad ktorý je jednoducho použiteľný a prehladný. Jednotlivé stránky su písané prevažne v HTML, ale pri formuláry pre vyhľadávanie a pridávanie záznamu bol použitý JavaScript kvôli dynamickému počtu prvkov. Formuláre majú tlačidlo “Add filed”, ktoré pridá nový prvok.

Prepojenie Formulárov a Backend-u

Pre správne fungovanie aplikácie je potrebné prepojiť formuláre s backend-om, inak povedané, uložiť hodnoty z formulárov do Java premenných. Tieto metódy sú implementované v HomeController a CategoriesController. Pre formuláre s dynamickým počtom prvkov bola použitá trieda org.springframework.util.MultiValueMap<String, String>, čo je vpodstate predpripravné Map<String, List<String>>, obohatené o pomocné metódy.
