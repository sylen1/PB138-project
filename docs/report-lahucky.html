XML database setup

Pri výbere XML databázy som sa rozhodol pre eXist-db z dôvodu, že je open source a práca s ňou bola ukázaná na cvičeniach, teda s ňou bol oboznámený každý člen tímu.

Nasledovalo vytvorenie rozumnej štruktúry XML databázy. Po konzultácii môjho prvotného návrhu s tímom vznikla súčasná forma, ktorú je možné nájsť napríklad v súbore sample-database.xml

XMLDatabaseManager implementation

Moja druhá úloha bola implementovať interface XMLDatabaseManager, ktorý obsahuje všetky metódy potrebné pre komunikáciu webovej aplikácie s XML databázou.

Zo začiatku som plánoval využiť XQJ - XQuery API for Java, ale vyskytli sa problémy pri XQuery dotazoch, ktoré mali vykonávať zmeny v databáze.

Rozhodol som sa teda pre prechod na XML:DB API, v ktorom fungovali aj problémové dotazy. S touto úlohou mi pomáhal aj zvyšok tímu, konkrétne:

  • Matej Lukeš implementoval metódu searchMedia,

  • Kristýna Pekárková implementovala findMaximumMediaId,

  • Ľudovít Labaj upozornil na chyby spojené s rozlišovaním veľkých a malých písmen
